# LOGBOT AI

An AI-powered IT Helpdesk Chatbot built using Flask, Gemini AI, and Incident Management Data.

## Features

- AI-powered chatbot
- Incident search from CSV data
- Ticket creation system
- Admin dashboard
- Chat history storage using SQLite
- Gemini AI integration
- Ollama fallback support

## Technologies Used

- Python
- Flask
- Gemini AI
- Pandas
- SQLite
- HTML
- CSS
- JavaScript
